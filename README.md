# Websocket Client by Golang

## Dependencies
- [Gorilla Websocket](github.com/gorilla/websocket)

## Howo to run
```
$ HOSTNAME=[YOUR HOSTNAME] PROTOCOL=[Protocol Name ws or wss] go run main.go
```
Note: `HOSTNAME` Parameter will input without ```wss://``` or ```ws://```
